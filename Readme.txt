***Project On Calculator using Reverse Polish Notation Using C Language.

***Installation Process:

	1.First using the Shell Command Line in Terminal, tunnel to the project folder via “cd” (change directory) command.

	2.Using Vim Editor : write command “vim filename.c”. Enter “I” , to insert anything in vi editor .
	Enter “Esc” Enter “:” write “wq” ,Enter.
		
	3.Using Compiler : For c code write “gcc - o filename filename.c” , Enter After compiling a runnable file will be created .

	4.Run : write “./filename” , Enter.
	
	5.Insert your input in reverse polish notation.
